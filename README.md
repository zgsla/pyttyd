# Pyttyd
一个Python开发的 web ssh 工具

### 安装

```commandline
$ pip install pyttyd
```

### 启动服务

```commandline
$ pyttyd
```

浏览器访问 [http://127.0.0.1:8221](http://127.0.0.1:8221)
